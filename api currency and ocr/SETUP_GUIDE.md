# API, Currency & OCR - Expense Reimbursement

## 🚀 Quick Setup Guide

### 1. Clone Repository
```bash
git clone https://github.com/lakshyagoyal284/Reimbursement-management.git
cd Reimbursement-management
```

### 2. Install Dependencies
```bash
pip install -r expense_reimbursement/requirements.txt
```

### 3. Install Tesseract OCR
- **Windows:** Download from [UB-Mannheim Tesseract](https://github.com/UB-Mannheim/tesseract/wiki)
- **Ubuntu:** `sudo apt-get install tesseract-ocr`
- **macOS:** `brew install tesseract`

### 4. Test Functionality
```bash
python demo_expense_functions.py
```

### 5. Install in Odoo
Copy `expense_reimbursement` folder to your Odoo addons directory and install the module.

## ✅ Features Working
- 💱 Currency Conversion (USD → EUR, etc.)
- 🌍 Country Currency Detection (US → USD, Germany → EUR)
- 🧾 OCR Receipt Processing (Extract amount, date, merchant)
